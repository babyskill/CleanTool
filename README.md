# CleanTool & 垃圾清理

# 注意
本工程运行需要系统签名和MTK平台支持
1.系统签名原因：清理缓存的接口需要系统签名，清理会出现程序强退
2.MTK平台的原因：基于MTK平台的Media本地数据库，如果放到高通平台上会出现数据库字段不一致导致扫描程序强退

本项目旨在提供一个本地快速扫描的思路，可以不需运行，看看代码即可
